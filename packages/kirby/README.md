# ianhobbsmedia/codey-design-system (Kirby plugin)

The Kirby layer: reusable snippets, block renderers, blueprints (blocks, fields,
sections), generic templates, and models. Installs via Composer into any Kirby
site.

Register each promoted piece in `index.php`. Keep site-specific content out —
only reusable system parts belong here (see the invariant in
[../../docs/ARCHITECTURE.md](../../docs/ARCHITECTURE.md)).

Populated in **Phase 3** of [../../docs/ROADMAP.md](../../docs/ROADMAP.md).
