# @codey/css

The CSS primitives: grid + container system (12-col, subgrid alignment), the
off-screen / off-page / menu-stack / push-stack overlay families, effects, base
elements, typography, and forms.

Framework-agnostic — no Kirby, no site content. Consumes `@codey/tokens` and
references tokens only; it never hardcodes colours, spacing, or type sizes.

Authored in Stylus, compiled to `dist/codey.css`. Populated in **Phase 2** of
[../../docs/ROADMAP.md](../../docs/ROADMAP.md).
