# @codey/tokens

Foundation layer: colour palettes, spacing, and the Utopia fluid type/space
scale, exposed as CSS custom properties (and, once built, a JSON mirror for
tooling).

Everything else in the design system references these tokens rather than
hardcoding values. To rebrand a site, override the tokens — the primitives that
consume them stay untouched.

Populated in **Phase 1** of [../../docs/ROADMAP.md](../../docs/ROADMAP.md).
